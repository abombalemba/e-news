# Взаимодействие компонентов

## Сбор новости

Ядро инициирует проверку источника. Модуль выполняет запрос к целевому сайту, извлекает данные и возвращает результат. Ядро получает результат, сравнивает с сохраненным состоянием источника. Если обнаружены новые данные, ядро обновляет состояние в хранилище и отправляет новость в брокер для рассылки.

## Доставка новости

Каналы рассылки постоянно слушают брокер. При появлении сообщения каждый канал получает новость. Канал запрашивает из хранилища список подписчиков и их настройки фильтрации. Для каждого пользователя проверяется, соответствует ли новость его интересам. Если соответствует, новость отправляется. Если пользователь исключил данную категорию или источник, новость пропускается.

## Управление подписками

Пользователь взаимодействует с ботом, отправляя команды. Бот обрабатывает команды и обновляет настройки пользователя в хранилище. Команды могут включать добавление или удаление исключений, просмотр текущих настроек, получение справки.
